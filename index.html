How to Use QTI-Former

1

Format Your Document

Write questions as normal paragraphs. List options below them. Mark the correct answer by making it bold, italic, prefixing with *, or adding {{ANS}}.

2

Review & Edit

Drag and drop your .docx file above. Review the parsed questions interactively. Click text to fix typos and use the radio buttons to adjust correct answers.

3

Import to Canvas

Click Download, then go to your Canvas Course > Settings > Import Course Content. Select QTI .zip file and upload your new package!
